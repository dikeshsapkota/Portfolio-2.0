## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/dikeshsapkota/Portfolio-2.0.git
```

Navigate to the project

```bash
cd Portfolio-2.0
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
